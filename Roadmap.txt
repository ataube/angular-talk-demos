Roadmap
========

--> Source: http://angularjs.org

1. What is Angular?
====================

a) A framework, but much more lightweight than a typical framework 
b) 100% JavaScript, 100% client side and compatible with both desktop and mobile browsers
c) Browser Support: Safari, Chrome, Firefox, Opera, IE8, IE9 and mobile browsers (Android, Chrome Mobile, iOS Safari).
d) File size < 29KB (compressed)


2. Setup / Hello World / Show Cases

d) Filters
